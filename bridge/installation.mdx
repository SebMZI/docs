## Installation

---

<Steps>
  <Step title="Download">
    Download the package on <a href="https://portal.cfx.re" target="_blank">Cfx.re</a>
  </Step>
  <Step title="Installation">
    - Install `smzi_bridge` and ensure the resource in your `server.cfg`
    - Make sure the resource is started after the resources used in the bridge and before the resources using the bridge 
  </Step>
  <Step title="Configuration">
    You don't have any configuration to do, except if one of the resources you use is not in the bridge
  </Step>
</Steps>
