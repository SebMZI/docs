## Installation

---

<Steps>
  <Step title="Download">
    Download the package on <a href="https://portal.cfx.re" target="_blank">Cfx.re</a>
  </Step>
  <Step title="Installation">
    - Install `smzi_bridge` and ensure the resource before `smzi_kiosk` 
    - Install `smzi_kiosk` and ensure it in your `server.cfg`
  </Step>
  <Step title="Configuration">
    Configure the script like you want using the configuraton documentation.
  </Step>
</Steps>

## Requirements

- SMZI Bridge -- Available in the CFX package
