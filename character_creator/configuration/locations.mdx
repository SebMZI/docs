---
title: "Locations"
---

Choose the character location where the first character customisation will be made and the location for the first ever spawn of your players.

```LUA
    Config.locations = {
        ["spawn"] = vector4(-1044.461548, -2749.437256, 21.360474, 331.653534),
        ["character_custom"] = vector4(-811.674744, 175.186813, 76.728882, 104.881889)
    }
```
