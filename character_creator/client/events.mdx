## ToggleCharacterCreatorNUI

Open the Character Creator - Identity creation NUI

```LUA
    --- @param state boolean # Toggle
    --- @param characterCreatorType string # "creation" or nil
    TriggerClientEvent("smzi_charactercreator:client:toggleCharacterCreatorNUI", state, characterCreatorType)
```

## ToggleSkinMenu

Open the Character Creator - Character Appearance NUI

```LUA
    --- @param state boolean # Toggle
    --- @param characterCreatorType string # "creation" or nil
    TriggerClientEvent("smzi_charactercreator:client:toggleSkinMenu", state, characterCreatorType)
```
